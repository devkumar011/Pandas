1. Data Creation Methods (Data Banana ya Load Karne ke Liye)
Method	English Explanation	Hinglish Explanation

pd.read_excel('file.xlsx')	Reads data from an Excel file.	Excel sheet ka data pandas mein import karta hai.
pd.read_json('file.json')	Reads JSON file.	JSON format ka data load karta hai.

pd.concat([df1, df2])	Joins multiple DataFrames vertically or horizontally.	Do ya zyada DataFrames ko jodta hai (upar-neeche ya side-by-side).
pd.merge(df1, df2)	Combines DataFrames using common columns (like SQL join).	Common column ke base par do DataFrames ko merge karta hai.
